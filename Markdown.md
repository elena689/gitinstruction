## Ссылки

Для создания ненумерованного списка перед строками ставь звездочку и пробел
```
Например:
* Добавить элемент 1
* Добавить элемент 2
* Добавить элемент 3
```
Для создания нумерованного списка просто перед строкой писать номер и точку.
```
Например:
1. 1й элемент нумерованного списка
2. 2й элемент списка
```
