# Инструкция по работе с Markdown

## Выделение текста
Для того, чтобы выделить текст курсивом необходимо обрамить его *
```
Пример: *курсив*
```
Чтобы выделить текст полужирным, необходимо обрамить **
```
Пример: **полужирный**
```
